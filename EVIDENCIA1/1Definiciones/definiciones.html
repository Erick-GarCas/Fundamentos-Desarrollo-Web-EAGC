Etiquetas de Encabezados (H1-H6)

Definición: Se utilizan para crear títulos y subtítulos jerárquicos en el documento.

Ejemplo visual:

Encabezado nivel 1

Encabezado nivel 2

Encabezado nivel 3

Encabezado nivel 4

Encabezado nivel 5

Etiquetas: <h1>, <h2>, <h3>, <h4>, <h5>, <h6>

Etiqueta P (Párrafo)

Definición: Define un párrafo de texto. Automáticamente agrega espacios antes y después del contenido.

Ejemplo visual:

Este es un párrafo de ejemplo. Los párrafos son bloques de texto que se separan automáticamente con espacios.

Etiqueta: <p>

Etiqueta HR (Línea horizontal)

Definición: Crea una línea horizontal que sirve como divisor temático entre secciones.

Ejemplo visual:

Etiqueta: <hr>

Etiqueta BR (Salto de línea)

Definición: Inserta un salto de línea simple sin crear un nuevo párrafo.

Ejemplo visual:

Primera línea
Segunda línea
Tercera línea

Etiqueta: <br>

Etiquetas STRONG y B (Texto en negrita)

Definición: STRONG indica importancia semántica, B aplica formato visual de negrita.

Ejemplo visual:

Texto normal, texto importante y texto en negrita

Etiquetas: <strong>, <b>

Etiqueta I (Cursiva)

Definición: Aplica formato de cursiva al texto contenido.

Ejemplo visual:

Texto normal y texto en cursiva

Etiqueta: <i>

Etiqueta U (Subrayado)

Definición: Subraya el texto contenido.

Ejemplo visual:

Texto normal y texto subrayado

Etiqueta: <u>

Etiqueta OL (Lista ordenada)

Definición: Crea una lista numerada donde el orden de los elementos es importante.

Ejemplo visual:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Etiqueta: <ol>

Etiqueta UL (Lista no ordenada)

Definición: Crea una lista con viñetas donde el orden no es importante.

Ejemplo visual:

  • Elemento con viñeta
  • Otro elemento
  • Último elemento

Etiqueta: <ul>

Etiqueta LI (Elemento de lista)

Definición: Define un elemento individual dentro de una lista ordenada o no ordenada.

Ejemplo visual:

  • Este es un elemento de lista

Etiqueta: <li>

Etiqueta MARK (Texto marcado)

Definición: Resalta o marca texto como si fuera con un marcador fluorescente.

Ejemplo visual:

Texto normal con texto marcado para destacar

Etiqueta: <mark>

Etiqueta SMALL (Texto pequeño)

Definición: Reduce el tamaño del texto, generalmente para texto secundario o legal.

Ejemplo visual:

Texto normal y texto pequeño

Etiqueta: <small>

Etiqueta EM (Énfasis)

Definición: Indica énfasis semántico en el texto, generalmente se muestra en cursiva.

Ejemplo visual:

Texto normal con énfasis importante

Etiqueta: <em>

Etiqueta CITE (Cita)

Definición: Marca el título de una obra citada (libro, artículo, película, etc.).

Ejemplo visual:

Mi libro favorito es Cien años de soledad de Gabriel García Márquez

Etiqueta: <cite>

Etiqueta Q (Cita corta)

Definición: Marca una cita corta en línea, automáticamente agrega comillas.

Ejemplo visual:

Como dijo Einstein: La imaginación es más importante que el conocimiento

Etiqueta: <q>

Etiqueta BLOCKQUOTE (Cita en bloque)

Definición: Marca una cita larga que se muestra como un bloque indentado.

Ejemplo visual:

Esta es una cita en bloque que generalmente se usa para citas largas. Se muestra indentada y separada del texto principal.

Etiqueta: <blockquote>

Etiqueta ABB (Abreviación)

Definición: Marca una abreviación o acrónimo, puede incluir el texto completo en el atributo title.

Ejemplo visual:

La OMS es una agencia de la ONU

Etiqueta: <abbr>

Etiqueta CODE (Código)

Definición: Marca fragmentos de código de programación con fuente monoespaciada.

Ejemplo visual:

Para imprimir en JavaScript usa: console.log("Hola mundo");

Etiqueta: <code>

Etiqueta SUB (Subíndice)

Definición: Muestra texto como subíndice, útil para fórmulas químicas o matemáticas.

Ejemplo visual:

La fórmula del agua es H2O

Etiqueta: <sub>

Etiqueta SUP (Superíndice)

Definición: Muestra texto como superíndice, útil para exponentes o referencias.

Ejemplo visual:

E = mc2 es la ecuación de Einstein

Etiqueta: <sup>

Etiqueta DEL (Texto eliminado)

Definición: Marca texto que ha sido eliminado del documento, se muestra tachado.

Ejemplo visual:

El precio original era $100 ahora $80

Etiqueta: <del>

Etiqueta IMG (Imagen)

Definición: Inserta una imagen en el documento. Requiere el atributo src.

Ejemplo visual:

Etiqueta: <img>

Etiqueta IFRAME (Marco en línea)

Definición: Embebe otro documento HTML dentro del documento actual.

Ejemplo visual:

Etiqueta: <iframe>

Etiqueta VIDEO (Video)

Definición: Embebe contenido de video en la página web con controles nativos.

Ejemplo visual:

Etiqueta: <video>

Etiqueta DETAILS (Detalles)

Definición: Crea un widget de revelación donde el usuario puede mostrar/ocultar información adicional.

Ejemplo visual:

Haz clic para ver más detalles

Este contenido se muestra cuando el usuario hace clic en el resumen.

Puede contener cualquier elemento HTML.

Etiqueta: <details>
